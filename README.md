## Installation


Setting up your development environment on your local machine :

```
$ git clone https://github.com/AsadullohRuziev/laravel-task.git
$ cd laravel-task
$ cp .env.example .env
$ composer install
$ npm install

```
## .env

```
DB_CONNECTION=mysql
DB_DATABASE=laravel-task
```

All following commands must be run inside the M:

```
$ npm run dev
$ php artisan serve
```
